Navigation and Links

Navigation and links are integral components of web development, allowing users to explore and interact with content on your website.

When navigating this website, you've clicked on buttons that direct you to your desired web page. Understanding how to create effective navigation is crucial for seamless user experience.

HTML Navigation

In HTML, navigation is often achieved through the <a> tags, representing links to different pages. The href attribute specifies the target page's file name or an external URL.

Caution: Not all links of a document should be inside a <nav> element. Links, such as those within the content of a page, may not necessarily be part of the main navigation and might not belong inside a <nav>.

Example

A set of navigation links:

<nav>
  <a href="/html/">HTML</a> |
  <a href="/css/">CSS</a> |
  <a href="/js/">JavaScript</a> |
  <a href="/python/">Python</a>
</nav>

Here is an example of this website's navigation:

<nav>
    <ul>
        <li><a href="indexx.html">Home</a></li>
        <li><a href="SettingUp.html">Setting Up</a></li>
        <li><a href="HTMLBasics.html">HTML Basics</a></li>
        <li><a href="CSSFundamentals.html">CSS Fundamentals</a></li>
        <li><a href="formBasics.html">Form Basics</a></li>
        <li><a href="navigation.html">Navigation and Links</a></li>
    </ul>
</nav>
