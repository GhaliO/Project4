CSS Fundamentals

In this section, you will learn about the fundamentals of CSS coding concepts. CSS is used for styling HTML documents and enhancing the visual presentation of web pages. CSS goes hand in hand with your HTML code, allowing you to control the layout of multiple web pages and the overall design of your document. If you want to change the color, font, or add a text box to your web pages, CSS allows you to do that.

Why Use CSS?

CSS is used to style and design your web page, including the layout. CSS also allows you to customize the screen sizes for different devices that will be viewing your web pages. For this website for example, I simply wrote my texts and structured them in HTML and designed my webpage using CSS in a separate file.

CSS Example

Example:

/* CSS Style */
body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
    font-size: 20px;
}

Try it yourself

Selectors and Declarations

CSS works by selecting HTML elements and applying style declarations to them. Here's a basic example:

Example:

/* CSS Style */
body {
    font-family: 'Arial', sans-serif;
    background-color: #f8f8f8;
    color: #333;
}

/* Selecting an HTML element */
h1 {
    color: #3498db;
}

In the example above, the body element is selected, and styles such as font-family, background-color, and color are applied. Additionally, the h1 element is selected with a specific color.

Box Model

The CSS box model describes the structure of every HTML element as a rectangular box with properties like margin, border, padding, and content.

Example:

/* CSS Box Model */
.example-box {
    width: 200px;
    height: 100px;
    margin: 20px;
    border: 2px solid #3498db;
    padding: 10px;
}

The example above creates a box with a width of 200 pixels, a height of 100 pixels, a margin of 20 pixels, a border of 2 pixels solid, and padding of 10 pixels.
