Key Definitions

  1. HTML: Stands for HyperText Markup Language. Used for structuring content on the web.
  2. CSS: Stands for Cascading Style Sheets. Used for styling and layout of HTML elements.
  3. Tag: An HTML element enclosed in angle brackets, such as <tagname>.
  4. Element: An element in HTML is a complete unit formed by opening and closing tags along with their content, such as <div>...</div>.
  5. Selector: In CSS, a selector is a pattern used to select and style one or more HTML elements, for instance, targeting all paragraphs (p).
  6. Attribute: Additional information provided within the opening tag of an HTML element (e.g., <img src="example.jpg" alt="Description">).
  7. Property: In CSS, a property, like color or font-size, represents a style aspect applied to an element to define its appearance.
  8. Value: The value, such as “18p” for font size or “red” for color, specifies how a particular style property should be applied to an element.
