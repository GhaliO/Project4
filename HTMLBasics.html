Understanding Basic HTML Structure

In this section, you will be shown some basic HTML structure examples. This page is for you to understand the basic sections and structure of HTML code. Donâ€™t worry if there are tags you have never learned about yet!

HTML is the programming language for creating the structure of a web page. Here are the components that you need to know to understand how it works:

  • All HTML documents have to start with the declaration: <!DOCTYPE html>. This simply represents the document type and allows browsers to identify and display your web pages.
  • The HTML document itself begins with <html> and ends with </html>.
  • The part of the HTML document that will actually be shown on your page is between <body> and </body>.

Example:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Try it yourself

Important concepts to understand:

  • Tag structure: HTML uses a system of tags to define the structure of a document. Tags are enclosed in angle brackets, such as <tagname>.
  • Elements: Tags can contain content and form HTML elements. Elements define the structure of a web page, such as headings (<h1>), paragraphs (<p>), and links (<a>). We will use these elements in the CSS code file later on to style the texts in between them!
  • Attributes: Tags can have attributes that provide additional information about the element. Attributes are added to the opening tag (<tagname attribute="value">).
  • Document Structure: An HTML document typically consists of a <head> section (containing meta-information) and a <body> section (containing the content).

Common HTML Elements:

Here are some common HTML elements:

  • <p>: Defines a paragraph.
  • <h1> to <h6>: Define header tags for headings.
  • <a>: Creates hyperlinks.
  • <img>: Embeds images.
  • <ul>, <ol>, <li>: Create unordered and ordered lists.
  • <div>: Defines a division or a section in an HTML document.

Example Code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My First Web Page</h1>
    <p>This is a paragraph of text.</p>
    <img src="example.jpg" alt="An example image">
</body>
</html>
